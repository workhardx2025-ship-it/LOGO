# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
npm run install-all
```

Это установит зависимости для корневого проекта, backend и frontend.

## Шаг 2: Настройка OpenAI API

1. Перейдите в папку `server`
2. Скопируйте `.env.example` в `.env`:
   ```bash
   cd server
   copy .env.example .env
   ```
   (На Linux/Mac: `cp .env.example .env`)

3. Откройте файл `server/.env` и добавьте ваш OpenAI API ключ:
   ```
   OPENAI_API_KEY=sk-your-api-key-here
   PORT=3001
   ```

   Получить API ключ можно на https://platform.openai.com/api-keys

## Шаг 3: Запуск приложения

Из корневой директории проекта:

```bash
npm run dev
```

Это запустит одновременно:
- Backend сервер на http://localhost:3001
- Frontend приложение на http://localhost:3000

## Шаг 4: Использование

1. Откройте браузер и перейдите на http://localhost:5173
2. Нажмите "Добавить ученика" и введите ФИО
3. Выберите ученика и создайте для него урок
4. Добавьте слова в урок (можно указать URL картинок)
5. Нажмите "Начать урок" и следуйте инструкциям

## Примечания

- При первом запуске автоматически создастся база данных SQLite в `server/database.sqlite`
- Для работы распознавания речи нужен доступ к микрофону браузера
- Временные аудио файлы сохраняются в `server/temp/` и автоматически удаляются

## Решение проблем

**Ошибка "OpenAI API key not configured"**
- Убедитесь, что файл `server/.env` существует и содержит правильный API ключ

**Микрофон не работает**
- Проверьте разрешения браузера на доступ к микрофону
- Убедитесь, что используете HTTPS или localhost (некоторые браузеры требуют HTTPS для доступа к микрофону)

**Ошибка при установке зависимостей**
- Убедитесь, что у вас установлен Node.js версии 16 или выше
- Попробуйте удалить `node_modules` и установить заново
